# 📊 Instamart Grocery Data Analysis

Welcome to a complete Data Analytics Case Study focused on Instamart sales and delivery performance.
This project analyzes a real-world business scenario using **Power BI** and **Power Query** to provide actionable insights for rapid grocery delivery operations.

## 🚀 Project Objective

The goal of this project is to analyze Blinkit's sales performance and provide actionable insights to help the business:
* Track total revenue and average order values.
* Understand customer purchasing behavior across product categories.
* Monitor sales performance across different outlet sizes and tiers.
* Evaluate average customer ratings to identify areas for service improvement.

## 🧰 Tech Stack

* **Power BI** – Interactive dashboard creation & data visualization
* **Power Query** – Data cleaning, processing, and transformation
* **DAX** – Custom metric calculations (KPIs)
* **Excel** – Raw data storage and initial exploration

## 📈 Key Analysis Performed

* Key Performance Indicators (Total Sales, Average Sales, Total Items, Average Rating).
* Sales distribution by item fat content (Low Fat vs. Regular).
* Performance comparison across Outlet Sizes (Small, Medium, High) and Locations (Tier 1, 2, 3).
* Item type popularity analysis (e.g., Fruits & Veggies, Snack Foods, Household).
* Historical sales trends based on the outlet establishment year.

## 📊 Power BI Dashboard

The interactive dashboard provides:
* An executive summary of overall sales performance.
* Dynamic filtering by outlet location, size, and specific item types.
* Custom tooltip visualizations and drill-down capabilities.
* Detailed matrix visuals comparing multiple metrics simultaneously.

## ⚠️ Note on Dataset

The dataset used for this project is a sample meant to reflect real-world rapid delivery metrics and is included in this repository.

## 🎯 Business Impact

This analysis helps stakeholders:
* Optimize inventory management based on high-demand product categories.
* Identify top-performing outlet configurations to replicate success in new locations.
* Target specific item types to improve overall average customer ratings.
* Make data-driven decisions regarding regional expansion.
